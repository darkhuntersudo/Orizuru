# 🌸 Orizuru: The Botanical QR Architect
**Orizuru** is not just another QR generator—it’s where data meets digital nature. While most QR codes are static and utilitarian, Orizuru transforms your links, contact info, and data into fully customized brand assets, paired with a unique, procedurally generated **3D Blossom Tree**.

## ✨ Features
 * **Total Customization:** Modify colors, gradients, eye shapes, and data patterns.
 * **Brand Integration:** Easily embed logos and custom frames into your QR codes.
 * **Procedural 3D Art:** Every QR code acts as a "seed." Our engine generates a unique **3D Blossom Tree** based on the specific data and parameters of your code.
 * **High-Res Exports:** Download your QR codes in SVG, PNG, or PDF formats, and capture your 3D tree in 4K.
 * **Interactive Viewer:** A WebGL-powered viewer to rotate and explore your generated blossom tree in real-time.
## 🛠️ How It Works
Orizuru uses a proprietary algorithm to map the data strings within a QR code to biological growth variables.
> **The Logic of the Bloom:**
> The complexity of your URL or data affects the branch density, while the color palette of your QR code influences the hue of the blossoms. No two trees are exactly alike.
> 
### Tech Stack
 * **Frontend:** React / Three.js (for the 3D rendering)
 * **QR Engine:** Custom logic for SVG manipulation
 * **Styling:** Tailwind CSS
 * **Animation:** Framer Motion
## 🚀 Getting Started
### Prerequisites
 * Node.js (v18 or higher)
 * npm or yarn
### Installation
 1. **Clone the repository**
   ```bash
   git clone https://github.com/darkhuntersudo/orizuru.git
   
   ```
 2. **Install dependencies**
   ```bash
   cd orizuru
   npm install
   
   ```
 3. **Run the development server**
   ```bash
   npm run dev
   
   ```

## 🎨 Usage
 1. **Input Data:** Enter your URL or text in the generator.
 2. **Design:** Use the sidebar to tweak the QR aesthetics (corners, colors, logo).
 3. **Watch it Grow:** Observe the 3D Blossom Tree evolve in the preview pane as you change your data.
 4. **Export:** Save your QR for print or web, and download the 3D tree as a digital collectible or background.
